# Financi_UI

Financi_UI is a modern financial mobile app UI built with Flutter.  
The project focuses on creating a clean, simple, and user-friendly banking/finance interface.

The app includes authentication screens, password recovery flow, cards dashboard, statistics screen, and user profile UI.

## Features

- Welcome screen
- Login screen
- Register navigation
- Continue as guest option
- Forgot password screen
- OTP verification screen
- Create new password screen
- Password changed success screen
- Home dashboard
- Financial cards carousel
- Send money and pay bill actions
- Statistics chart screen
- All cards screen
- User profile screen
- Bottom navigation bar
- Clean and modern UI design

## Getting Started

Follow these steps to run the project locally.

### 1. Clone the repository

```bash
git clone https://github.com/kero12-eg/Financi_UI.git
```

### 2. Go to the project folder

```bash
cd Financi_UI
```

### 3. Install dependencies

```bash
flutter pub get
```

### 4. Run the app

```bash
flutter run
```

---

## Build Android APK

```bash
flutter build apk
```

---

## Build iOS

```bash
flutter build ios
```

> Building for iOS requires macOS and Xcode.
