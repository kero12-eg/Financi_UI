# Financi_UI

Financi_UI is a modern financial mobile app UI built with Flutter.  
The project focuses on creating a clean and user-friendly banking/finance interface with authentication screens, card management, statistics, and profile pages.

## App Preview

The app includes multiple screens such as login, password recovery, OTP verification, home dashboard, statistics chart, cards page, and user profile.

## Features

- Welcome screen
- Login screen
- Register navigation
- Continue as guest option
- Forgot password screen
- OTP verification screen
- Create new password screen
- Password changed success screen
- Home dashboard
- Financial cards carousel
- Send money and bill payment actions
- Statistics chart screen
- All cards screen
- User profile screen
- Bottom navigation bar
- Clean and modern UI design
